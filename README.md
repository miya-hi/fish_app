# README

# 開発言語
- Ruby 2.6.5
- Rails 5.2.5

# 就業Termの技術
- device
- お気に入り機能

# カリキュラム外の技術
- chartkick

# 実行手順
$ git clone git@github.com:miya-hi/fish_app.git  
$ cd fish_app  
$ bundle install  
$ yarn install  
$ rails db:create  
$ rails db:migrate  
$ rails db:seed_fu  
$ rails s
